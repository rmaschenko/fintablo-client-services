# TEMPLATES — реестр Excel-шаблонов anti-ICP

12 шаблонов из библиотеки Финтабло отдаются anti-ICP-сегменту. URL **автоматически собираются** из публичной папки Я.Диска — в коде ничего вшивать руками не нужно.

---

## Как работает выдача (TL;DR)

В [`quiz/js/templates.js`](quiz/js/templates.js):

```js
const PUBLIC_KEY = 'https://disk.yandex.ru/d/NPG3nHemZ743oQ';
const FILE_SUFFIX = '. Чтобы пользоваться шаблоном, создайте копию.xlsx';

function buildUrl(fileName) {
  return PUBLIC_KEY + '?path=' + encodeURIComponent('/' + fileName + FILE_SUFFIX);
}
```

При клике на карточку шаблона пользователь попадает в UI Я.Диска с **открытым предпросмотром файла** (302 redirect внутри домена `disk.360.yandex.ru`). Дальше — кнопка «Скачать» в UI Я.Диска. Авторизация не требуется.

---

## Что нужно делать при изменениях

### Если папка переехала в другое место
Поменять `PUBLIC_KEY` в [templates.js:16](quiz/js/templates.js#L16) — все 12 ссылок пересоберутся автоматически.

### Если файл переименовали
Поменять `file` в `REGISTRY` ([templates.js:34](quiz/js/templates.js#L34)) у соответствующего шаблона. Например, если `[ФИНТАБЛО] ОПиУ` стал `[ФИНТАБЛО] Отчёт о прибылях`:

```js
opiu: tpl(
  'opiu',
  'ОПиУ',
  '...',
  '[ФИНТАБЛО] Отчёт о прибылях'  // ← новое имя
),
```

### Если поменяется суффикс
Сейчас все файлы заканчиваются на `. Чтобы пользоваться шаблоном, создайте копию.xlsx`. Если суффикс поменяют — поменять `FILE_SUFFIX` в [templates.js:18](quiz/js/templates.js#L18).

## Как проверить что URL рабочие

```bash
# Все 12 ссылок должны отвечать HTTP 302 → disk.360.yandex.ru
node -e "global.window={};require('./quiz/js/templates.js');
  const r=global.window.Templates.REGISTRY;
  Object.keys(r).forEach(k=>console.log(k+'\\t'+r[k].url))" \
| while IFS=$'\t' read id url; do
  echo "$(curl -sI -A 'Mozilla/5.0' "$url" | head -1) $id"
done
```

## Альтернативные подходы (зачем выбран этот)

| Подход | Минус | Плюс |
|---|---|---|
| **Прямые публичные ссылки на каждый файл** (генерация в UI Я.Диска) | Нужно 12 ручных публикаций; новый файл — снова руками | Один клик — сразу скачивание |
| **PHP-прокси** (`api/template.php?id=opiu`) | Нужен PHP-бэкенд; не работает на статическом GitHub Pages | Контроль логирования, downloads counter |
| **Я.Диск API + OAuth** для генерации public_url | Требует токен пользователя, риски ротации | Полностью автоматизировано |
| ✅ **`?path=` в публичной папке** | UX в 2 клика (попадает в UI Я.Диска, потом «Скачать») | Без бэкенда, без ручной работы, само пересобирается при переезде папки |

Выбран `?path=` — оптимальный trade-off для текущего MVP без CRM-маркетолога.
