# money-profit · «Деньги есть — прибыли нет»

Диагностический микросервис Финтабло. Перехватывает ICP-клиента в момент острой боли «деньги на счёте есть, а прибыли нет» и через персонализированный AHA-момент мотивирует оставить заявку.

**Прод (план):** `https://fintablo.ru/tools/money-profit/`

## Целевая конверсия

Baseline CR АБ Яндекс.Директа на главную = 0,07%. Цель сервиса — **0,25–0,50% CR АБ** за счёт встроенной квалификации и персонализированного расчёта разрыва.

## Дизайн-система

IBM Plex Sans (UI) + IBM Plex Mono (числа, tabular-nums) + Literata (заголовки). DS-токены определены в [css/style.css](./css/style.css) — цвета, типографика, spacing (4px-шаг), радиусы, тени.

## UX-паттерны квиза

- Иллюзия прогресса с первого шага (прогресс-бар виден сразу).
- Один вопрос на экран, ≤ 4 варианта ответа (парадокс выбора Шварца).
- Микро-награды после каждого ответа (социальное доказательство, инсайты).
- Контакт — **только** после последнего вопроса, до показа результата.
- «Назад» доступно всегда.
- Эффект Зейгарник: `beforeunload` при попытке закрыть на 50%+ прогрессе.
