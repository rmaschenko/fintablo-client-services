# Fintablo Client Services

Клиентские PLG-микросервисы Финтабло. Каждый сервис — полностью автономная статическая страница со своими шрифтами, стилями, JS и `api/lead.php` для передачи лидов в AmoCRM.

## Сервисы

| Сервис | Статус | URL (прод) | Описание |
|---|---|---|---|
| [`money-profit/`](./money-profit/) | 🚧 В разработке | `fintablo.ru/tools/money-profit/` | «Деньги есть — прибыли нет». Диагностика разрыва между остатком на счёте и реальной прибылью. |

## Архитектурные правила

- **Каждый сервис автономен.** Общих `shared/` папок нет. Шрифты, стили, компоненты дублируются — это осознанный tradeoff ради изоляции: изменение в одном сервисе не ломает другой.
- **Всё работает локально в браузере.** Расчёты на JS, никаких внешних API в логике. PHP только для отправки лида в AmoCRM.
- **Шрифты только локальные** (152-ФЗ, без Google Fonts / IBM CDN). SIL Open Font License 1.1 — IBM Plex Sans/Mono + Literata.
- **Работает на обычном российском shared-хостинге** (PHP 7.4+) без токенов в коде — секреты через `.env` (в `.gitignore`).

## Локальный запуск

```bash
cd money-profit
python3 -m http.server 8000
# → http://localhost:8000
```

Для `api/lead.php` нужен локальный PHP-сервер:
```bash
php -S localhost:8000
```

## Деплой

Пока только локально и GitHub. Деплой на прод (`fintablo.ru/tools/...`) — вручную пользователем, FTP не автоматизируется.
